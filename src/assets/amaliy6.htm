·        O’rnatilgan Windows 2008 Server operatsion tizimi  elementlari  yig’ilgan disk.

 

I.  NAZARIY QISM

  Windows 2008 Serverni o‘rnatishni boshlashdan avval, o‘rnatish dasturi (Setup) birinchi navbatda operatsion tizimni o‘zini yuklashi kerak. Buning uchun distributiv kompakt-diskdan yoki o‘rnatiladigan yuklanuvchi diskyetlardan yuklanishi kerak.

Agar kompyuterda OS Windows NT 2008 Server o‘rnatilgan bo‘lsa, Winnt  komandasini (agar kompyuter MS-DOS tomonidan boshqarilib ishlasa) yoki Winnt 32 (Agar kompyuter Windows NT 2008 Server tomonidan boshqarilib ishlasa) 1386 katalogidan (Intel protsyessorlar bazasidagi kompyuterlar uchun). Kitobni bu bo‘limda o‘rnatish jarayoniga asosiy e’tiborni qaratilgan, bunda kompyuterni yuklanishi kompakt-diskdan yoki o‘rnatiladigan disketlarda bajariladi.

 Agar siz qachondir OT ni o‘rnatgan bo‘lsangiz, bu sizga oson bo‘ladi, chunki intyerfyeys o‘rnatish dasturi oldingi vyersiyalarga o‘xshash. O‘rnatish dasturining taklif qilinadigan ro‘yxatdan kerakli opsiyani ajratish uchun kursorni yuritadigan tugmalardan foydalaning, kerakli opsiya tanlash uchun “Enter” tugmasi ishlatiladi, hech qanday harakat qilmasdan o‘rnatish dasturini tamomlash uchun - <F3> tugmasi ishlatiladi.

Bu oynada operatsion tizim tilini, sana va vaqt formatini va klaviatura tilini tanlab Next ni bosamiz.

Bu erda Install Now ni bosamiz.

Bu oynada Opersion tizim kalitini kiritishimiz lozim. Agar sizda OT ni litsenzion kaliti mavjud bo‘lsa uni shu erda kiritasiz. Bu mashg‘ulot uchun OTni vaqtinchalik litsenziyasini ishlatamiz, ya’ni, OTdan kalit kiritmasdan 30 kun davomida foydalanishimiz mumkin.  Demak, Automatically activate Windows when I’m online tugmachasini belgilanmagan holga keltiramiz va xech qanday kalit kiritmasdan Next ni bosamiz.

Bu joyda yuklanayotgan Windowsni versiyasi tanlanadi, agarda bundan oldingi oynada kalitni kiritganimizda, shu kalitga qarab Windowsni versiyasi avtomatik tarzda tanlangan bo‘lar edi. Lekin kalitni kiritmaganimiz uchun Windowsni qaysi versiyasini yuklash kerakligini o‘zimiz tanlashimiz kerak bo‘ladi. Xozir Windows Server 2008 Enterprise (Full Installation)ni tanlaymiz. Server Coreda esa Windows grafik interfeysi mavjud emas. YA’ni, Server Coreda tartib ida ishlashda foydalanuvchi server bilan faqat buyruq ya’ni komandalar kiritib ishlaydi. Bu imkoniyat Windows Server 2008 da kiritildi.

Windowsni versiyasini tanlab bo‘lganimizdan so‘ng, I have selected the edition of Windows that I purchased tugmachasini belgilab qo‘yamiz va Next ni bosamiz.

Endi agarda biz xozir OTni Windows Server 2003 OT ishlayatgan kompyuterga ustanovka qiladigan bo‘lganimizda edi, bu erda  Upgrade tugmasini bosish imkoni mavjud bo‘lar edi. Lekin xozir biz Windows Server 2008 ni noldan ustanovka qilayapmiz, shuning uchun Custom (advanced) ni bosamiz.

Administratorni parolini 2 marta kiritamiz va Enter  klavishasini bosamiz.

Tarmokni sozlash uchun Configure Networkingni bosamiz. Bu erda kompyuterimizda mavjud bo‘lgan tarmoq ulanishlari kelritirlan. Tarmoq kartasini xossalari (Svoystva yoki Properties) oynasini ochamiz.

Bu oynada Properties ni bosamiz.

Windows Server 2008 OTda tarmoq bilan ishlash uchun IPv6, ya’ni TCP/IP protokolining oltinchi versiyasi avval-boshdan yuklangan  va ishlatish uchun belgilab qo‘yilgan bo‘ladi. Endi agar siz tarmog‘ingizda IPv6 ni ishlatmaydigan bo‘lsangiz, uni o‘chirib qo‘ying, chunki undan foydalanmasangiz uni bu erda turishidan ma’no yuq. Agar mabodo keyinroq kerak bo‘lib qolsa, uni shu erdan osongina yoqib qo‘yishingiz mumkin bo‘ladi. Xozir TCP/IPv4ni sozlaymiz...uni tanlab turib Properties tugmasini bosamiz.

Tarmoq kartamizga statik IP adres beramiz. Keling IP adresga 192.168.5.120 ni kiritamiz, tabni boscak tarmoq maskasi avtomatik tarzda to‘ldiriladi. Default gateway joyiga tarmog‘imizni marshrutizatorining va Prefered DNS Server joyiga DNS serverni IP adreslari kiritiladi, misol uchun 192.168.5.1 deb kiritamiz. OK ni bosamiz, va Close ni bosamiz. Ishonchim komilki bu narsalarni ko‘plaringiz ming marotalab qilgansiz.
